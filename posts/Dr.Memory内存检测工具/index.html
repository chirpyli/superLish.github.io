Dr.Memory内存检测工具

Jul 21, 2016 2016-07-21T13:25:00+08:00 by lisheng
Updated Feb 17 2021-02-17T21:28:19+08:00 3 min

Dr. Memory 是一个开源免费的内存检测工具，它能够及时发现内存相关的编程错误，比如未初始化访问、内存非法访问以及内存泄露等。与 Valgrind 类似，可以直接检查已经编译好的可执行文件。用户不用改写被检查程序的源代码，也无须重新链接第三方库文件，使用起来非常方便。

安装

在Linux下安装参考Installing on Linux.

#include <stdio.h>

	// 对空指针指向的内存区域写，会发生段错误
	int *null_ptr = NULL;
	*null_ptr = 10;

	return 0;
}
